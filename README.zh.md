# 每日计划

### 简介

通过拖拽的方式对添加的任务进行状态划分

可以通过键盘的上下键或者WS键进行项目、计划、设置页面上下切换

在计划页面，可以通过键盘的左右键或者AD键在不同的项目之间进行切换。

在项目页面可以直接鼠标点击想要打开的项目，快速跳转到对应的计划页面。
